Welcome to OnEarth

This demo is customized with pre-confgured endpoints and sample imagery. For more information, visit https://github.com/nasa-gibs/onearth

Endpoint Configuration

A "test" endpoint is included for basic configuration and debugging. Two endpoints are included for profiling performance: "profiler" and "profiler_reproject".

Four additional endpoints are configured for the following EPSG projections: 4326 (geographic), 3857 (webmercator), 3413 (arctic), and 3857 (antarctic). Sample imagery is not included for these endpoints.

The OnEarth Test Client will load any layers configured in the GetCapabilities for each respective endpoint
